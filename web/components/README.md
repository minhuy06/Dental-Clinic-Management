# Thư mục Components (Thành phần Dùng chung)

## Chứa các đoạn mã HTML/JSP nhỏ lẻ, mang tính chất lặp đi lặp lại trên nhiều trang khác nhau. Việc tách ra giúp chúng ta chỉ cần viết code 1 lần và tái sử dụng ở mọi nơi.

## Thành phần
Ví dụ các file nên đặt ở đây:
- `header.jsp` (Thanh menu điều hướng trên cùng)
- `footer.jsp` (Phần chân trang, thông tin liên hệ phòng khám)
- ...
