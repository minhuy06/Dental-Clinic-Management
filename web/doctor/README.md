# Thư mục Giao diện Bác sĩ

## Thư mục này CHỈ chứa các trang giao diện (file `.jsp`) dành riêng cho chức năng của Bác sĩ.

## Các trang
- Trang danh sách hàng đợi.
- Trang khám lâm sàng.

## Phân quyền
Các trang trong thư mục này cần được bảo vệ. Chỉ những tài khoản đăng nhập với vai trò 'Bác Sĩ' mới được phép truy cập.
