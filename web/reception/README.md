# Thư mục Giao diện Lễ tân.

## Thư mục này CHỈ chứa các trang giao diện (file `.jsp`) dành riêng cho chức năng của Lễ tân.

## Các trang
- Trang quản lý hồ sơ bệnh nhân & thu ngân.
- Trang quản lý Lịch hẹn.

## Phân quyền
Các trang trong thư mục này cần được bảo vệ. Chỉ những tài khoản đăng nhập với vai trò 'Lễ tân' mới được phép truy cập.
